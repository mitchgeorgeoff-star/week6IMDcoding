# Week 6 IMD CSS Project

This project includes a simple HTML page and stylesheet for learning CSS functions and typography.

## Files

- `index.html` — main page showing a CSS example
- `styles.css` — stylesheet with responsive typography and CSS function usage

## How to use

1. Open `index.html` in your browser.
2. Edit `styles.css` to experiment with `clamp()`, `calc()`, `var()`, and gradient colors.
